Introduction to HTML

HTML (HyperText Markup Language): The standard language for creating web pages and web applications.

Structure: HTML uses a system of tags to enclose different parts of the content. Tags are composed of a name enclosed in angle brackets.

Basic Structure of an HTML Document

!DOCTYPE html: Declaration that defines the document type and version of HTML.

html: The root element that contains all other elements.

head: Contains meta-information about the document (e.g., title, links to stylesheets).

title: Sets the title of the document, which appears in the browser tab.

body: Contains the content of the document (e.g., text, images, links).
